📚 Bridge the Books
A modern web platform designed to connect people who want to donate books with those who need them. This project focuses on making knowledge accessible by bridging the gap between donors and learners.
🚀 Project Overview
Bridge the Books is a full-stack web application built to simplify book sharing and donations. It provides an intuitive interface where users can:
📖 Donate books
🔍 Request needed books
🤝 Connect with donors and recipients
🌐 Access a user-friendly and responsive platform
🛠️ Tech Stack
This project is built using modern web technologies:
⚡ Vite
⚛️ React
🟦 TypeScript
🎨 Tailwind CSS
🧩 shadcn/ui
📂 Getting Started (Run Locally)
Follow these steps to run the project on your system:
Bash
# Clone the repository
git clone https://github.com/your-username/your-repo-name.git

# Navigate into the project folder
cd your-repo-name

# Install dependencies
npm install

# Start the development server
npm run dev
